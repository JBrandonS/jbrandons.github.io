# The Early Universe: Machine Learning and the CMB

Exploring the early universe's conditions that led to the large-scale structure we observe today.

## Overview

By analyzing cosmic microwave background (CMB) data using Machine Learning models, we aim to uncover the statistical properties of primordial fluctuations to levels never before achieved. This research combines advanced computational techniques with fundamental cosmology.

**Technologies:** Python, TensorFlow/Keras, Cython, CAMB, Pixell, LensPyx, HealPy
