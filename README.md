## Welcome to GitHub Pages

You can use the [editor on GitHub](https://github.com/JBrandonS/jbrandons.github.io/edit/master/README.md) to maintain and preview the content for your website in Markdown files.

Whenever you commit to this repository, GitHub Pages will run [Jekyll](https://jekyllrb.com/) to rebuild the pages in your site, from the content in your Markdown files.

## Local Development

To run this site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000` in your browser.

## Customization

Edit the `_config.yml` file to update your personal information, social media links, and site settings.
