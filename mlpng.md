# Primordial Non-Gaussianity

Machine Learning Approaches to Detecting Non-Gaussian Signatures in the Early Universe.

## Overview

Primordial non-Gaussianity is a key signature of the physics governing the early universe and inflation. Using advanced machine learning techniques, we can extract subtle non-Gaussian signals from cosmological data with unprecedented sensitivity.
